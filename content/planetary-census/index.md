# Planetary Census

Population data is pulled from the FIO API and cached for one hour. The table shows the most recent population report for each planet. Click any column header to sort.

Be aware that this FIO pull is 60mb + of json, so it takes a while. "Region" listed is the currency of the planet and may not actually reflect the real location or control of that planet.
